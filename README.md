# 💻 Meu Portfólio Profissional

Bem-vindo ao repositório do meu site pessoal! Este projeto foi desenvolvido para apresentar minha identidade como Desenvolvedor Web e destacar meus principais serviços e projetos.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Design moderno, layouts com Flexbox e responsividade total (Mobile First).
* **JavaScript (Vanilla):** Lógica para o carrossel de imagens, FAQ interativo e máscaras/validações de formulário.
* **EmailJS:** Integração para recebimento de mensagens diretamente no meu e-mail sem a necessidade de um back-end complexo.

## 🛠️ Funcionalidades

* **Carrossel Dinâmico:** Destaque visual na página inicial.
* **Seção de Serviços:** Detalhamento de planos para Landing Pages e Sites Institucionais.
* **Filtros e Máscaras:** Validação em tempo real de Nome, E-mail e Telefone para garantir a qualidade dos leads recebidos.
* **Design Responsivo:** Navegação otimizada para computadores, tablets e smartphones.
